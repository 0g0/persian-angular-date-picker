Installation

See index.html file in project.

  1. Add angular.js
  2. Add moment.js and moment-jalali.js
  3. Add persian-angular-date-picker.css
  4. Add persian-angular-date-picker.js
  5. Update your app defination: 
    var app = angular.module('date-picker-demo', ['bizbank.angular']);
  6. Use the directive:
    <persian-angular-date-picker ng-model="model" date-change="change()" ></persian-angular-date-picker>
    ng-model: required
    date-change: optional
